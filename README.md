# ChatGPT 问题队列助手

一个 Chrome 浏览器扩展，为 ChatGPT 添加问题队列，支持自动提问和捕获上下文。

## ✨ 主要功能

* **问题队列**: 在 ChatGPT 页面添加一个可拖动的悬浮面板，用于管理问题列表。
* **自动发送**: 当 ChatGPT 空闲时，自动从队列中取出下一个问题并发送。
* **上下文捕获**: 在页面上选中任意文本，点击 `+` 按钮，即可将其作为上下文附加到你的问题中。
* **拖拽排序**: 在队列面板中，可以直接拖动问题卡片来调整发送顺序。
* **自动保存**: 你的问题队列会自动保存，刷新页面也不会丢失。

## 📦 如何安装 (手动加载)

1.  下载本项目的所有文件，并解压到一个文件夹中（例如 `chatgpt-queue`）。
2.  打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/` 并回车。
3.  确保右上角的 **“开发者模式”** 开关已打开。
4.  点击左上角的 **“加载已解压的扩展程序”**。
5.  选择你刚才解压的那个文件夹 (`chatgpt-queue`)。
6.  安装完成！访问 ChatGPT 页面即可开始使用。
